# Portfolio Informatika

Situs portofolio sederhana, responsif, dan beranimasi. Dibuat dengan HTML, CSS, dan JavaScript tanpa build step.

## Fitur
- Tentang Saya
- Portofolio (filter kategori)
- Skill (progress bar)
- Sertifikat (kartu)
- Kontak (form ke Formspree)

## Cara Jalankan
Buka file `index.html` langsung di browser.

## Personalisasi Cepat
- Ubah teks nama, bio, dan fakta di `index.html` (bagian `hero` dan `about`).
- Ganti link sosial dan email di `index.html`.
- Ubah daftar proyek, kategori, dan tag di `#portfolio`.
- Ubah nilai persentase skill (inline style `--v`) di `#skills`.
- Ganti gambar (gunakan URL atau taruh file dan ubah `src`).
- Atur action Formspree pada tag `form` (ganti `your-id`).
